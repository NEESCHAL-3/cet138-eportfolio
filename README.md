## 🌟 About the Project
This portfolio serves as an educational and interactive guide to understanding the layers of full-stack web development. Instead of static text, it combines structural documentation with live code demos, interactive components, and responsive grid layouts, highlighting how front-end and back-end layers communicate.

### ❓ Core Q&A

#### 1. What is this website about?
This website is a Full Stack Development ePortfolio designed for the **CET138 Full Stack Development (Assignment 1)** course. It provides a structured, interactive walk-through of the foundational technologies behind web applications: Semantic HTML, Responsive CSS, Bootstrap Framework, and dynamic JavaScript (ES6+).

#### 2. What will it do?
It displays theoretical explanations side-by-side with working frontend and backend code simulators. It provides live demonstrations of layout grids, responsive media queries, CSS variables, framework components, local state management (via a task tracker), and asynchronous network communication.

#### 3. What can be done on it?
Visitors can:
*   **Modify themes dynamically**: Adjust the background color of CSS cards using the color picker to see custom CSS properties in action.
*   **Manage checklist tasks**: Add, complete, and delete tasks in a local checklist powered by DOM manipulation.
*   **Simulate API fetch requests**: Fetch mock JSON data asynchronously to see how databases, servers, and clients communicate.
*   **Trigger framework components**: Interact with a responsive grid and launch an interactive Bootstrap modal.
*   **Navigate responsively**: Test the site across different breakpoints (desktop, tablet, mobile) to see responsive layout adjustments.

## 📚 Topics Covered

### 1. What is Full Stack Development?
*   Explains the request/response cycle between the client (Browser), front-end templates, server-side code (Back end), and the database.
*   Includes a visual representation of the **Simple Request Flow** and a sample JSON API response.

### 2. Semantic HTML
*   Demonstrates structural layout using accessible and semantic HTML5 elements (`<article>`, `<header>`, `<figure>`, `<figcaption>`, and `<footer>`).
*   Includes an accessible form structure and a formatted tasks table.

### 3. Responsive CSS & Design Systems
*   Features a custom layout designed using **CSS Grid** and **CSS Flexbox**.
*   Utilizes a media query breakpoint at `850px` to shift the layout dynamically for mobile, tablet, and desktop devices.
*   Includes a **real-time Color Picker** showing how JavaScript can interact with CSS custom variables to update themes.

### 4. Bootstrap Framework
*   Showcases the integration of a utility-first and component-based CSS framework.
*   Demonstrates Bootstrap's responsive grid structure (`row`, `col-md-6`), alert notifications, badge pills, and an **interactive Modal window** triggered via Bootstrap's bundle.

### 5. Interactive JavaScript (ES6+)
*   Features a **live task manager checklist** that supports adding, marking, and deleting tasks dynamically without refreshing the page.
*   Demonstrates event listening, DOM modification, array manipulation, and input validation.
*   Includes a `fetch()` API simulator that pulls mock JSON data asynchronous.

## 🛠️ Technologies Used
*   **HTML5** — Semantic layout, forms, tables, and skip links for web accessibility.
*   **CSS3** — Custom properties (variables), transitions, hover animations, CSS Grid, and media queries.
*   **JavaScript (ES6)** — DOM API, event handlers, local arrays, and API fetch calls.
*   **Bootstrap v5.3.3** — Responsive columns, flex utilities, badges, alerts, and modal scripts.
*   **Google Fonts** — Styled using the modern, clean *Inter* typeface.
